7-дәріс: Python Операторлары

Операторлар айнымалылар мен мәндерге әртүрлі операцияларды орындау үшін қолданылады.

Мысалы, екі мәнді қосу үшін "+" операторын қолданамыз:

Мысал:


                print(10 + 5)

Python операторлары келесі топтарға бөлінеді:

    Арифметикалық операторлар Меншіктеу операторлары Салыстыру операторлары Логикалық операторлар Идентификация операторлары Тиістілік операторлары Биттік операторлар

Python Арифметикалық операторлары

Арифметикалық операторлар сандармен жұмыс істеп, негізгі математикалық операцияларды орындайды

Оператор Атауы Мысалы
+ Қосу x + y
- Алу x - y
* Көбейту x * y
/ Бөлшекпен бөлу x / y
% Қалдықты табу x % y
** Дәрежеге шығару x ** y
// Бүтін бөлу x // y

Python Меншіктеу операторлары

Бұл операторлар айнымалыға мәндерді меншіктеу үшін қолданылады:

Оператор Мысалы Сол сияқты
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 3 x = x - 3
*= x *= 3 x = x * 3
/= x /= 3 x = x / 3
%= x %= 3 x = x % 3
//= x //= 3 x = x // 3
**= x **= 3 x = x ** 3
&= x &= 3 x = x & 3
|= x |= 3 x = x | 3
^= x ^= 3 x = x ^ 3
>>= x >>= 3 x = x >> 3
<<= x <<= 3 x = x << 3

Python Салыстыру операторлары

Бұл операторлар екі мәнді салыстыру үшін қолданылады:

Оператор Атауы Мысалы
== Тең x == y
!= Тең емес x != y
> Үлкен x > y
<< Кіші x < y
>= Үлкен немесе тең x >= y
<= Кіші немесе тең x <= y

Python Логикалық операторлары

Логикалық операторлар шарттарды біріктіру үшін қолданылады:

Оператор Сипаттамасы Мысалы
and Екі шарт та орындалса, True қайтарады x < 5 and x < 10
or Бір шарт орындалса, True қайтарады x < 5 or x < 4
not Шартты терістейді, False қайтарады, егер True болса not(x < 5 and x < 10)

Python Идентификация операторлары

Бұл операторлар объектілердің бірдей жад орнында орналасқанын тексеру үшін қолданылады:

Оператор Сипаттамасы Мысалы
is Егер екі айнымалы бірдей объект болса, True қайтарады x is y
is not Егер екі айнымалы әртүрлі объект болса, True қайтарады x is not y

Python Тиістілік операторлары

Бұл операторлар тізбектің объект ішінде бар-жоғын тексеру үшін қолданылады:

Оператор Сипаттамасы Мысалы
in Егер объект ішінде тізбек болса, True қайтарады x in y
not in Егер объект ішінде тізбек болмаса, True қайтарады x not in y

Python Биттік операторлары

Бұл операторлар екілік сандарды бит бойынша салыстыру үшін қолданылады:

Оператор Атауы Сипаттамасы
& AND Егер екі бит те 1 болса, 1 қайтарады
| OR Егер екі биттің біреуі 1 болса, 1 қайтарады
^ XOR Егер тек бір бит 1 болса, 1 қайтарады
~ NOT Барлық биттерді инверттейді
<< Солға жылжу Биттерді солға жылжытып, нөлдерді қосады
>> Оңға жылжу Биттерді оңға жылжытып, сол жақтағы биттерді көшіріп қосады
