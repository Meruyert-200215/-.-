2-дәріс: Python-дағы айнымалылар

Айнымалылар — бұл деректердің мәндерін сақтау үшін арналған контейнерлер.

Python-да айнымалыларды жасау

Басқа бағдарламалау тілдерінен айырмашылығы, Python-да айнымалыны жариялау үшін арнайы команда жоқ.

Мысал

Айнымалы алғаш рет мән тағайындалған кезде жасалады.


                        x = 5
                        y = "Лаура"
                        print(x)
                        print(y)

Айнымалыларды нақты бір типпен көрсетудің қажеті жоқ және олар орнатылғаннан кейін типін өзгертуі мүмкін.

Басқа бағдарламалау тілдерінде әдетте айнымалылардың типін көрсету қажет! Python-да бұл міндетті емес! 


                    x = 4  # x бүтін сан (типі int)
                    x = "Раушан"  #  `x` жол типінде (типі `str`).
                    print(x)

Python-да Кастинг

Егер айнымалының деректер типін көрсету қажет болса, мәндерді түрлендіру арқылы оны жасауға болады.

Мысал


                x = str(3) # x '3' болады
                y = int(3) # y 3 болады
                z = float(3) # z  3.0 болады
                print(x)
                print(y)
                print(z)

Python-да тип алу

Айнымалының типін type() функциясы арқылы алуға болады.

Мысал


            x = 5
            y = "Ернар"
            print(type(x))
            print(type(y))

Python: Бір жақты немесе қос жақты үтірлер?

Айнымалыларды бір жақты (') немесе қос жақты (") үтірлермен жариялауға болады.

Мысал


        x = "Айдар"
        # бірдей болады
        y = 'Айдар'
        print(x)
        print(y)

Python: Регистрге сезімталдық

Айнымалы атаулары регистрге сезімтал.

Мысал

Екі айнымалыны жасайық :


                a = 4
                A ="Татьяна"
                print(a) 
                print(a)
                # A қайта жазылмайды

Айнымалылардың атаулары

Айнымалы қысқа атауға ие болуы мүмкін (мысалы, x және y) немесе сипаттамалы атауға (мысалы, жас, көлік_аты, жалпы_көлем).

Python-дағы айнымалылардың атауларына қойылатын ережелер:

• Айнымалының атауы әріптен немесе астыңғы сызу (_) таңбасынан басталуы керек.
• Айнымалының атауы саннан басталмауы керек.
• Айнымалының атауы тек әріптерді, сандарды және астыңғы сызу таңбасын қамти алады (A-z, 0-9, _).
• Айнымалылардың атаулары регистрге сезімтал (мысалы, жас, Жас және ЖАС — бұл үш түрлі айнымалы).

Мысал

Рұқсат етілген айнымалы атаулары: 


     myvar = "Жігер"
     my_var = "Жігер"
     _my_var = "Жігер"
     myVar = "Жігер"
     MYVAR = "Жігер"
     myvar2 = "Жігер"

Мысал

Рұқсат етілмеген айнымалы атаулары: 


    2myvar = "Жігер"
    my-var = "Жігер"
    my var = "Жігер"

Айнымалы атауларының регистрге сезімталдығын есте сақтаңыз

Бірнеше сөзден тұратын айнымалы атаулары

Бірнеше сөзден тұратын айнымалы атаулары оқуға қиын болуы мүмкін. Оларды оқуды жеңілдету үшін бірнеше әдістерді қолдануға болады.

Camel Case (Түйе регистрі)

Әрбір сөз, бірінші сөзден басқа, бас әріппен басталады:

Мысал


        myVariableName = "Жігер"

Pascal Case (Паскаль регистрі)

Әрбір сөз бас әріппен басталады:

Мысал


        MyVariableName = "Жігер"

Snake Case (Жылан регистрі)

Әрбір сөз астыңғы сызу (_) таңбасымен бөлінеді:

Мысал


        my_variable_name  = "Жігер"

Python-да бірнеше мәндер

Бірнеше айнымалыға бірнеше мәндерді тағайындау
Python сізге бірнеше айнымалыға бір жолда мәндер тағайындауға мүмкіндік береді:

Мысал


              x, y, z = "Алма", "Банан", "Шие"
            print(x)
            print(y)
            print(z)

Ескерту:

Айнымалылар саны мен мәндер санының сәйкес келетініне көз жеткізіңіз, әйтпесе қате аласыз.

Бір мәнді бірнеше айнымалыға тағайындау
Сіз бірдей мәнді бірнеше айнымалыға бір жолда тағайындай аласыз: 


                x = y = z = "Алмұрт"
                print(x)
                print(y)
                print(z)

Жинақты ашу

Егер сізде тізім, кортеж немесе басқа жиын түрінде мәндер жиынтығы болса, Python осы мәндерді айнымалыларға бөліп алуға мүмкіндік береді. Бұл процесс жинақты ашу деп аталады.

Мысал

Жинақты ашу тізім арқылы: 


                        fruits = ["Апельсин", "Банан", "Шие"]   
                        x, y, z = fruits 
                        print(x) 
                        print(y) 
                        print(z)

Python-да айнымалыларды шығару

Айнымалыларды шығару үшін Python-ның print операторы жиі қолданылады.

Мысал:

Текст пен айнымалыны біріктіру үшін Python + таңбасын қолданады: 


                    x = "керемет"  
                    print("Python - бұл " + x)  
                    # Нәтиже: Python - бұл керемет

Айнымалыларды біріктіру

Сондай-ақ, + таңбасын бір айнымалыны екіншісіне қосу үшін пайдалануға болады:

Мысал: 


                x = "Python "  
                y = "керемет"  
                z = x + y  	
                print(z)  
                # Нәтиже: Python керемет

Сандар үшін + таңбасы математикалық оператор ретінде жұмыс істейді: 


        x = 5  
        y = 10  
        print(x + y)  
        # Нәтиже: 15

Егер сіз санды жолмен біріктіруге тырыссаңыз, Python қате береді: 


            x = 5  
            y = "Жігер"  
            print(x + y)  
            # Қате: TypeError: unsupported operand type(s) for +: 'int' and 'str'

Python-дағы жаһандық айнымалылар

Функциядан тыс жасалған айнымалылар (жоғарыдағы барлық мысалдардағы сияқты) жаһандық айнымалылар деп аталады. Жаһандық айнымалыларды функциялардың ішінде де, сыртында да қолдануға болады.

Мысал

Функциядан тыс айнымалы жасап, оны функция ішінде қолданыңыз: 


    x = "керемет"
    def myfunc():
        print("Python бұл " + x)
    myfunc()
    # Нәтиже: Python бұл керемет

Егер функция ішінде сол атаумен айнымалы жасасаңыз, ол локальды айнымалы болады және тек сол функцияның ішінде ғана қолданылады. Сол атаудағы жаһандық айнымалы өзгеріссіз қалады және өзінің бастапқы мәнін сақтайды.

Мысал 


                x = "керемет"

def myfunc():
    x = "ғажап"  # Локальды айнымалы
    print("Python бұл " + x)

myfunc()
print("Python бұл " + x)
# Нәтиже:
# Python бұл ғажап
# Python бұл керемет

Глобал (global) кілттік сөзі

Әдетте, функция ішінде жасалған айнымалылар локальды болып табылады және тек сол функцияның ішінде қолданылады. Егер сіз функция ішінде жаһандық айнымалы жасағыңыз келсе, global кілттік сөзін пайдалануға болады.

Мысал

Егер global кілттік сөзін қолдансаңыз, айнымалы жаһандық аймаққа тиесілі болады: 


    def myfunc():
    global x
    x = " ғажап"
  
  myfunc()
  
  print("Python бұл" + x)
  # Нәтиже: Python бұл ғажап

Глобал айнымалыларды өзгерту

Сондай-ақ, егер сіз жаһандық айнымалының мәнін функция ішінде өзгерткіңіз келсе, global кілттік сөзін пайдаланыңыз.

Мысал

Жаһандық айнымалының мәнін функция ішінде өзгерту: 


    x = "керемет"
def myfunc():
    global x
    x = "ғажап"
myfunc()
print("Python бұл " + x)
# Нәтиже: Python бұл ғажап
