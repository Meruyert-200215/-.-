3-дәріс: Python Деректер түрлері

Кірістірілген деректер түрлері

Бағдарламалауда деректер түрі маңызды ұғым болып табылады.
Айнымалылар әртүрлі деректер түрлерін сақтай алады, және әртүрлі түрлер әртүрлі әрекеттерді орындай алады.

Python тілінде әдепкі бойынша келесі деректер түрлері бар және олар келесі санаттарға бөлінеді:

Мәтін түрі: str
Сандық түрлер: int, float, complex
Тізбектік түрлер: list, tuple, range
Көрсету түрі: dict
Жиын түрлері: set, frozenset
Логикалық түр bool
Екілік түрлер bytes, bytearray, memoryview

Деректер түрін алу

Кез келген объектінің деректер түрін білу үшін type() функциясын пайдалануға болады.

Мысал

Айнымалы x-тің деректер түрін шығару: 


    x = 5
    print(type(x))

Python-да деректер түрін орнату

Python-да деректер түрі айнымалыға мән тағайындаған кезде автоматты түрде анықталады:

Мысал Деректер түрі
x = "Сәлем Әлем" str
x = 20 int
x = 20.5 float
x = 1j complex
x = ["алма", "банан", "шие"] list
x = ("алма", "банан", "шие") tuple
x = range(6) range
x = {"name": "Айдар", "age": 36} dict
x = {"алма", "банан", "шие"} set
x = frozenset({"алма", "банан", "шие"}) frozenset
x = True bool
x = b"Сәлем" bytes
x = bytearray(5) bytearray
x = memoryview(bytes(5)) memoryview

Белгілі бір деректер түрін орнату

Егер деректер түрін нақты көрсету қажет болса, келесі құрылымдық функцияларды пайдалануға болады:

Мысал Деректер түрі
x = str("Сәлем Әлем") str
x = int(20) int
x = float(20.5) float
x = complex(1j) complex
x = list(("алма", "банан", "шие")) list
x = tuple(("алма", "банан", "шие")) tuple
x = range(6) range
x = dict(name="Айдар", age=36) dict
x = set(("алма", "банан", "шие")) set
x = frozenset(("алма", "банан", "шие")) frozenset
x = bool(5) bool
x = bytes(5) bytes
x = bytearray(5) bytearray
x = memoryview(bytes(5)) memoryview
