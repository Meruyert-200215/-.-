10-дәріс: Python Функциялары,Лямбда,Массивтер, Мұрагерлік

Функцияны құру

Python-да функция def кілттік сөзі арқылы анықталады.

Мысал


                def my_function():
                  print('Функциядан сәлем')

Функцияны шақыру

Функцияны шақыру үшін оның атауын және соңынан дөңгелек жақшаларды қолданыңыз.

Мысал


                def my_function():
                  print('Функциядан сәлем')
                
                my_function()

Аргументтер

Функцияларға ақпаратты аргументтер түрінде бере аласыз. Аргументтер функция атауынан кейінгі дөңгелек жақшалар ішінде жазылады.

Мысал 


                
                def my_function(fname):
                  print(fname + ' Рефснес')
                
                my_function('Амир')
                my_function('Айнар')
                my_function('Лаура')

Параметрлер мен аргументтер

Параметр – бұл функцияны анықтау кезінде дөңгелек жақшалар ішінде көрсетілген айнымалы. Аргумент – бұл функция шақырылған кезде оған жіберілетін мән

Аргументтердің саны

Функция дұрыс аргументтер санына ие болуы керек.

Мысал 

               
                def my_function(fname, lname):
                  print(fname + ' ' + lname)
                
                my_function('Элина', 'Ернар')

Қалаған аргументтер, *args

Егер сіз функцияға қанша аргумент жіберілетінін білмесеңіз, параметр атауынан бұрын * қойыңыз.

Мысал 


                def my_function(*kids):
                  print('Ең кіші бала: ' + kids[2])
                
                my_function('Эсмира', 'Темирлан', 'Лера')

Кілттік сөз аргументтері, **kwargs

Егер функцияға жіберілетін кілттік сөз аргументтерінің санын білмесеңіз, параметр атауынан бұрын ** қойыңыз.

Мысал 


                
                def my_function(**kid):
                  print('Оның фамилиясы ' + kid['lname'])
                
                my_function(fname='ТОмар', lname='Айбар')

Әдепкі мән

Егер аргумент берілмесе, әдепкі мәнді пайдалану үшін параметрлерге әдепкі мән орнатуға болады.

Мысал 


                def my_function(country='Қазақстан'):
                  print('Мен ' + country + ' еліненмін')
                
                my_function('Россия')
                my_function('Индия')

Списокты аргумент ретінде беру

Функцияға кез келген дерек түрін, соның ішінде списктерді жіберуге болады.

Мысал 


                def my_function(food):
                  for x in food:
                    print(x)
                
                fruits = ['алма', 'банан', 'шие']                
                my_function(fruits)

Мәнді қайтару

Функция нәтижені қайтару үшін return операторын пайдаланады.

Мысал 


                def my_function(x):
                  return 5 * x
                
                print(my_function(3))
                print(my_function(5))
                print(my_function(9))

Функцияны бос қалдыру

Функция мазмұны болмаған жағдайда, қатені болдырмау үшін pass операторын пайдаланыңыз.

Мысал 


                
                def my_function():
                  pass

Рекурсия

Рекурсия – функцияның өзін шақыруы.

Мысал 



                def tri_recursion(k):
                  if k > 0:
                    result = k + tri_recursion(k - 1)
                    print(result)
                  else:
                    result = 0
                  return result
                
                print("Рекурсия нәтижесі:")
                tri_recursion(6)

Синтаксис

lambda аргумент : өрнек

Өрнек орындалып, нәтижесі қайтарылады.

Мысал

Аргументке 10 қосып, нәтижесін қайтару:


                x = lambda a: a + 10  
                print(x(5))  # Нәтиже: 15

Лямбда-функциялар кез келген мөлшердегі аргументтерді қабылдай алады:

Мысал

a аргументін b аргументіне көбейту:


        x = lambda a, b: a * b  
        print(x(5, 6))  # Нәтиже: 30

Мысал

a, b және c аргументтерін қосып, нәтижені қайтару:


        x = lambda a, b, c: a + b + c  
        print(x(5, 6, 2))  # Нәтиже: 13

Неге лямбда-функцияларды қолдану керек?

Лямбда-функцияның күші, әсіресе оны басқа функцияның ішінде анонимді функция ретінде қолданғанда көрінеді.
Мысалы, сізде бір аргумент қабылдайтын және ол аргументті белгісіз санға көбейтетін функция анықтамасы бар: 


        def myfunc(n):  
        return lambda a: a * n

Бұл функция анықтамасын әрқашан сіз жіберген санды екі еселейтін функция жасау үшін пайдаланыңыз:

Мысал 


        def myfunc(n):  
         return lambda a: a * n  

         mydoubler = myfunc(2)  
         print(mydoubler(11))  # Нәтиже: 22

Сол анықтаманы әрқашан сіз жіберген санды үш еселейтін функция жасау үшін қолданыңыз:

Мысал 


        def myfunc(n):  
        return lambda a: a * n  
      
      mytripler = myfunc(3)  
      
      print(mytripler(11))  # Нәтиже: 33

Немесе екі функцияны бір бағдарламада жасау үшін бірдей анықтаманы қолданыңыз:

Мысал 


        def myfunc(n):  
        return lambda a: a * n  
      
      mydoubler = myfunc(2)  
      mytripler = myfunc(3)  
      
      print(mydoubler(11))  # Нәтиже: 22  
      print(mytripler(11))  # Нәтиже: 33

Лямбда-функцияларды анонимді функция қысқа уақытқа қажет болған кезде пайдаланыңыз.

Массивтер бірнеше мәнді бір айнымалыда сақтау үшін қолданылады:

Мысал

Автомобильдердің аттарын қамтитын массив жасаңыз: 


        cars = ["Ford", "Volvo", "BMW"]

Массив деген не?

Массив — бұл бірнеше мәнді бір уақытта сақтай алатын арнайы айнымалы.Егер сізде элементтердің тізімі (мысалы, автомобильдер тізімі) болса, автомобильдерді жеке айнымалыларда сақтау келесідей көрінер еді:


        car1 = "Ford"  
        car2 = "Volvo"  
        car3 = "BMW"

Ал енді барлық көліктерді қарап, біреуін табу керек болса ше? Немесе сізде тек 3 көлік емес, 300 көлік болса ше?
Бұл жағдайда массив қолайлы шешім болады!
Массив бірнеше мәнді бір атаумен сақтай алады, және сіз индекспен мәндерге қол жеткізе аласыз.

Массив элементтеріне қол жеткізу

Массивтің элементіне индексті қолдану арқылы қол жеткізуге болады.

Мысал

Массивтің бірінші элементінің мәнін алу: 


        x = cars[0]

Мысал

Массивтің бірінші элементінің мәнін өзгерту: 


    cars[0] = "Toyota"

Массивтің ұзындығы

Массивтің ұзындығын (элементтер саны) қайтару үшін len() әдісін қолдануға болады.

Мысал 


        cars массивінің элементтер саны:
        x = len(cars)

Ескерту: Массивтің ұзындығы әрқашан массивтің ең үлкен индексінен бірге көп болады.

Массив элементтері бойынша цикл

Сіз for in циклы арқылы массивтің барлық элементтерін айналдыра аласыз.

Мысал

cars массивінің әрбір элементін шығару: 


        for x in cars:  
          print(x)

Массивке элемент қосу

Массивке элемент қосу үшін append() әдісін қолдануға болады.

Мысал

cars массивіне тағы бір элемент қосыңыз: 


        cars.append("Honda")

Массивтен элементті жою

Массивтен элементті жою үшін pop() әдісін қолдануға болады.

Мысал

cars массивінің екінші элементін жою: 


        cars.pop(1)

Сондай-ақ remove() әдісін элементті жою үшін қолдануға болады.

Мысал

Volvo мәні бар элементті жою: 


        cars.remove("Volvo")

Ескерту: remove() әдісі тек бірінші табылған мәнді жояды

Массивтің әдістері

Python-да тізімдер/массивтер үшін бірнеше ішкі әдістер бар.

Әдіс Сипаттама
append() Элементті тізімнің соңына қосады
clear() Тізімдегі барлық элементтерді өшіреді
copy() Тізімнің көшірмесін қайтарады
count() Көрсетілген мәннің тізімдегі санын қайтарады
extend() Тізімнің соңына басқа тізімді қосады
index() Көрсетілген мәннің бірінші индексын қайтарады
insert() Белгіленген орынға элемент қосады
pop() Белгіленген орындағы элементті жояды
remove() Көрсетілген мәннің бірінші кездескенін жояды
reverse() Тізімнің реті кері болады
sort() Тізімді сұрыптайды

Ескерту: Python-да тікелей массивтерді қолдау жоқ, бірақ тізімдер (lists) массивтер ретінде пайдаланылады.

Ата-ана класын жасау

Кез келген класс ата-ана класы бола алады, сондықтан синтаксис басқа кластарды жасау сияқты болады:

Мысал

Адам туралы класс жасау, firstname және lastname қасиеттері және printname әдісі бар:


        class Person:
          def __init__(self, fname, lname):
            self.firstname = fname
            self.lastname = lname
        
          def printname(self):
            print(self.firstname, self.lastname)

Person классынан объект жасау және printname әдісін орындау:


        x = Person("Акпаров", "Айбар")
        x.printname()

Бала класын жасау

Бала класты жасау үшін ата-ана класын параметр ретінде жіберіңіз:

Мысал

Person классынан мұрагерлік алатын Student класын жасау: 


        class Student(Person):
          pass

Ескерту: Егер класыңызға қосымша қасиеттер мен әдістерді қосқыңыз келмесе, pass кілттік сөзін қолданыңыз.
Енді Student класы Person классымен бірдей қасиеттер мен әдістерге ие болады.

Мысал

Student классынан объект жасап, printname әдісін орындаңыз: 


        x = Student("Акбарова", "Раушан")
        x.printname()

__init__() функциясын қосу

Енді біз бала класын жасадық, ол ата-ана классынан барлық қасиеттер мен әдістерді мұрагерлікке алады. Біз бала класына жаңа __init__() функциясын қосқымыз келеді (pass кілттік сөзінің орнына).
Ескерту: __init__() функциясы әрқашан жаңа объект жасалған кезде автоматты түрде шақырылады.

Мысал

Student классында __init__() функциясын қосу: 


   class Student(Person):
     def __init__(self, fname, lname):
       #қасиеттер мен т.б. қосу

Бала класына __init__() функциясын қосқанда, ол енді ата-ана класының __init__() функциясын мұрагерлікке алмайды.
Ескерту: Бала класының __init__() функциясы ата-ана класының __init__() функциясын қайта анықтайды.
Ата-ана класының __init__() функциясын сақтап қалу үшін, ата-ана функциясын шақыру керек:

Мысал 


class Student(Person):
def __init__(self, fname, lname):
Person.__init__(self, fname, lname)

Енді біз сәтті түрде __init__() функциясын қосып, ата-ана класынан мұрагерлікке алу мүмкіндігін сақтап қалдық, және біз __init__() функциясына функционалдық қосуға дайынбыз.

super() функциясын қолдану

Python-да super() функциясы да бар, ол бала класына ата-ана класынан барлық әдістер мен қасиеттерді мұрагерлікке алуға мүмкіндік береді:

Мысал 


        class Student(Person):
        def __init__(self, fname, lname):
          super().__init__(fname, lname)

super() функциясын қолданғанда, ата-ана класының атын қолданудың қажеті жоқ, ол автоматты түрде ата-ана класынан барлық әдістер мен қасиеттерді мұрагерлікке алады.

Қасиеттер қосу

Мысал

Student классына graduationyear деп аталатын қасиет қосу: 


        class Student(Person):
        def __init__(self, fname, lname):
          super().__init__(fname, lname)
          self.graduationyear = 2019

Төмендегі мысалда 2019 жыл айнымалы болуы керек және оны Student классын жасау кезінде объектілерге беру қажет. Оны жасау үшін __init__() функциясына тағы бір параметр қосу керек:

Мысал

year параметрін қосып, объектілер жасағанда дұрыс жылды көрсетіңіз: 


        class Student(Person):
          def __init__(self, fname, lname, year):
            super().__init__(fname, lname)
            self.graduationyear = year
        
        x = Student("Акбарова", "Раушан", 2019)

Әдістер қосу

Мысал

Student классына welcome әдісін қосу: 


        class Student(Person):
        def __init__(self, fname, lname, year):
          super().__init__(fname, lname)
          self.graduationyear = year
      
        def welcome(self):
          print("Қош келдіңіз", self.firstname, self.lastname, "класына", self.graduationyear)

Егер сіз бала класында ата-ана классындағы атпен бірдей әдіс қоссақ, ата-ана классындағы әдіс мұрагерлікке алынбайды, ол қайта анықталады.
