6-дәріс: Python Булев типі (True немесе False)

Булев мәндер екі жағдайды білдіреді: True немесе False.

Булев мәндердің қажеттілігі

Бағдарламалауда сізге жиі белгілі бір өрнектің True немесе False екенін білу керек болады.
Python кез келген өрнекті бағалап, осы екі нәтижені қайтарады:

Мысал: 


                print(10 > 9)  # True
                print(10 == 9) # False
                print(10 < 9)  # False

Шартты операторларда қолдану

if шарты орындалғанда, Python булев жауап қайтарады: True немесе False.

Мысал: 


                a = 200
                b = 33
                
                if b > a:
                  print("b a-дан үлкен")
                else:
                  print("b a-дан кіші")

Мәндер мен айнымалыларды бағалау

Python-ның bool() функциясы кез келген мәнді бағалап, True немесе False қайтара алады.

Мысал:

Жолдарды және сандарды бағалау: 


                print(bool("Сәлем"))  # True
                print(bool(15))       # True

Айнымалыларды бағалау: 


                x = "Сәлем" 
                y = 15
                print(bool(x))  # True
                print(bool(y))  # True

Көптеген мәндер "True" болып бағаланады

Көптеген мәндер True, егер олардың мазмұны бар болса

    Кез келген жол True, тек бос жолдан басқа. Кез келген сан True, тек 0-ден басқа. Кез келген тізім, жинақ, не сөздік True, тек бос болса ғана False.

Мысал:

Мына өрнектер True қайтарады: 


                print(bool("abc"))                 # True
                print(bool(123))                   # True
                print(bool(["apple", "banana"]))   # True

Кейбір мәндер "False" болып бағаланады

Бос мәндер мысалы: ((), [], {}, ""), сан 0, None, және "False" мәндері "False" болып бағаланады.

Мысал:

Мына өрнектер False қайтарады: 


                print(bool(False))   # False
                print(bool(None))    # False
                print(bool(0))       # False
                print(bool(""))      # False
                print(bool(()))      # False
                print(bool([]))      # False
                print(bool({}))      # False

Кейбір объектілер False болып бағалануы мүмкін, егер олар len функциясы арқылы 0 немесе "False" қайтаратын болса:

Мысал:


          class myclass():
                def __len__(self):
                   return 0
          myobj = myclass()
          print(bool(myobj))  # False

Функциялар булев мәндер қайтара алады

Сіз функцияларды құрып, олардың True немесе False қайтаратын нәтижелерін пайдалана аласыз.

Мысал:

Функцияны басып шығару: 


            def myFunction():
            return True
          print(myFunction())  # True

Булев жауап негізінде код орындау:

Мысал: 

            
                def myFunction():
                return True
                if myFunction():
                 print("Иә!")
                 else:
                  print("Жоқ!")

Python-да isinstance() сияқты кірістірілген функциялар бар, олар объектінің белгілі бір деректер түріне жататынын тексеруге қолданылады.

Мысал:

Объектінің бүтін сан екенін тексеру: 

            
                x = 200
                print(isinstance(x, int))  # True
