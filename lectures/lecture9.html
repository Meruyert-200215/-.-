9-дәріс: Python Операторлары : If / Else, While, For

Python шарттары және If операторы

Python математикадан таныс қарапайым логикалық шарттарды қолдайды:
• Тең: a == b
• Тең емес: a != b
• Кіші: a < b
• Кіші немесе тең: a <= b
• Үлкен: a > b
• Үлкен немесе тең: a >= b
Бұл шарттарды әртүрлі жолмен, көбінесе if операторларында және циклдарда қолдануға болады. If операторы if кілттік сөзі арқылы жазылады.

Мысал:

If операторы


        a = 33  
        b = 200  
        if b > a:  
          print("b a-дан үлкен")

Бұл мысалда біз a және b деген екі айнымалыны пайдаланамыз. If операторы арқылы b-нің a-дан үлкен екенін тексереміз. a == 33 және b == 200 болғандықтан, біз b-нің a-дан үлкен екенін білеміз және экранға «b a-дан үлкен» деген жазу шығарамыз.

Шегініс

Python кодтағы әрекет аймағын анықтау үшін шегіністерге (жол басындағы бос орындарға) сүйенеді. Басқа программалау тілдері мұны жиі фигуралық жақшалармен жасайды.

Мысал

Шегініссіз if операторы (қате тудырады): 


        a = 33  
        b = 200  
        if b > a:  
        print("b a-дан үлкен")  # қате аласыз

Elif

elif кілттік сөзі Python-ға «алдыңғы шарттар дұрыс болмаса, осы шартты байқап көр» деп айту тәсілі.

Мысал 


        a = 33  
        b = 33  
        if b > a:  
        print("b a-дан үлкен")  
        elif a == b:  
        print("a және b тең")

Бұл мысалда a == b, сондықтан бірінші шарт дұрыс емес, бірақ elif шарты дұрыс, сондықтан экранға «a және b тең» деген жазу шығарамыз.

Else

else кілттік сөзі алдыңғы шарттар дұрыс болмағанның бәрін ұстап алады.

Мысал 


    
        a = 200  
        b = 33  
        if b > a:  
          print("b a-дан үлкен")  
        elif a == b:  
          print("a және b тең")  
        else:  
          print("a b-дан үлкен")

Бұл мысалда a b-дан үлкен, сондықтан бірінші шарт дұрыс емес, сондай-ақ elif шарты да дұрыс емес. Сондықтан else шартына өтіп, «a b-дан үлкен» деген жазу шығарамыз.
Сіз else-ді elif-сіз де жаза аласыз:

Мысал 


     a = 200  
        b = 33  
        if b > a:  
          print("b a-дан үлкен")  
        else:  
          print("b a-дан үлкен емес")

Қысқа жазу әдісі If

Егер бір ғана операторды орындау қажет болса, оны if операторымен бір жолға орналастыруға болады.

Мысал 


        if a > b: print("a b-дан үлкен")

Қысқа жазу әдісі If ... Else

Егер сізде if және else үшін бір-бірден ғана оператор болса, оларды бір жолға жазуға болады:

Мысал 


        a = 2  
        b = 330  
        print("A") if a > b else print("B")

Бұл әдіс тернарлық операторлар немесе шартты өрнектер деп аталады. Көптеген else операторларын да бір жолға жазуға болады:

Мысал 


        a = 330  
        b = 330  
        print("A") if a > b else print("=") if a == b else print("B")

And

and кілттік сөзі логикалық оператор болып табылады және шартты операторларды біріктіру үшін қолданылады.

Мысал 


        a = 200  
        b = 33  
        c = 500  
        if a > b and c > a:  
          print("Екі шарт та дұрыс")

Or

or кілттік сөзі логикалық оператор болып табылады және шартты операторларды біріктіру үшін қолданылады.

Мысал 


         a = 200  
        b = 33  
        c = 500  
        if a > b or a > c:  
          print("Кем дегенде бір шарт дұрыс")

Ішкі If

Сіз if операторын басқа if операторының ішіне жаза аласыз, бұл ішкі if операторлары деп аталады.

Мысал 


        x = 41  
        if x > 10:  
          print("10-нан үлкен,")  
          if x > 20:  
            print("және 20-дан үлкен!")  
          else:  
            print("бірақ 20-дан үлкен емес.")

Өткізіп жіберу операторы

If операторлары бос болмауы керек, бірақ егер қандай да бір себеппен мазмұнсыз if операторыңыз болса, қатеден аулақ болу үшін pass операторын қосыңыз.

Мысал 


        a = 33  
        b = 200  
        
        if b > a:  
          pass

Python-да барлығы 2 цикл түрі бар:

• While циклі
• For циклі

While циклі

while циклы шарт дұрыс болғанша операторлар жиынын орындауға мүмкіндік береді.

Мысал

i 6-дан кіші болғанша оны басып шығарыңыз:


                i = 1  
                while i < 6:  
                  print(i)  
                  i += 1

Ескерту: i мәнін ұлғайтуды ұмытпаңыз, әйтпесе цикл мәңгі жалғасады.
while циклы тиісті айнымалылардың дайын болуын талап етеді. Осы мысалда біз цикл айнымалысы i-ге бастапқы мән ретінде 1 береміз.

break oператоры

break операторы арқылы біз циклды шарт дұрыс болса да тоқтата аламыз.

Мысал

i 3-ке тең болғанда циклдан шығыңыз:

   
                i = 1  
                while i < 6:  
                  print(i)  
                  if i == 3:  
                    break  
                  i += 1

continue oператоры

continue операторы ағымдағы итерацияны өткізіп жіберіп, келесіге көшуді қамтамасыз етеді.

Мысал

i 3-ке тең болғанда келесі итерацияға өтіңіз: 


                i = 0  
                while i < 6:  
                  i += 1  
                  if i == 3:  
                    continue  
                  print(i)

Else oператоры

else операторы арқылы шарт жалған болғанда код блогын бір рет орындауға болады.

Мысал

Шарт жалған болғанда хабарлама басып шығарыңыз: 


                i = 1  
                while i < 6:  
                  print(i)  
                  i += 1  
                else:  
                  print("i 6-дан кіші емес")

Бұл цикл басқа бағдарламалау тілдеріндегі for кілт сөзінен ерекшеленеді және көбінесе объектіге бағытталған тілдердегі итератор әдістері сияқты жұмыс істейді. for циклын пайдаланып, біз тізімдер, кортеждер, жиындар және т.б. әр элементі үшін операторлар жиынын бір рет орындай аламыз.

Мысал

Тізімдегі әрбір жемісті басып шығарыңыз: 


                fruits = ["алма", "банан", "шие"]  
                for x in fruits:  
                 print(x)

Ескерту: for циклы үшін индекстейтін айнымалыны алдын ала орнату қажет емес.

Жолдарды қайталау

Жолдар да итерирленетін объектілер болып табылады, яғни олар символдар тізбегін қамтиды.

Мысал

"банан" сөзіндегі әріптерді қайталаңыз:


                for x in "банан":  
                  print(x)

break операторы

break операторы арқылы циклды барлық элементтерді қарастырмай тұрып тоқтатуға болады.

Мысал

x "банан" болғанда циклды тоқтатыңыз:


                fruits = ["алма", "банан", "шие"]  
                for x in fruits:  
                  print(x)  
                  if x == "банан":  
                    break

Мысал

x "банан" болғанда циклды тоқтатыңыз, бірақ бұл жолы break операторын print алдында орналастырыңыз: 

                fruits = ["алма", "банан", "шие"]  
                for x in fruits:  
                  if x == "банан":  
                    break  
                  print(x)

Continue операторы

continue операторы ағымдағы итерацияны өткізіп жіберіп, келесісін жалғастырады.

Мысал

"банан" сөзін басып шығармаңыз: 

              
                fruits = ["алма", "банан", "шие"]  
                for x in fruits:  
                  if x == "банан":  
                    continue  
                  print(x)

range() функциясы

Белгілі бір код жиынтығын қайталап орындау үшін range() функциясын қолдануға болады.
range() функциясы сандар тізбегін қайтарады, әдепкі бойынша 0-ден басталады, 1-ге артып отырады және көрсетілген санмен аяқталады.

Мысал

range() функциясын қолданыңыз: 


                for x in range(6):  
                  print(x)

Ескерту : range(6) мәндері 0-ден 5-ке дейінгі сандарды қамтиды, бірақ 6-ны қоспайды.

Мысал

Бастапқы мәнді көрсетіңіз: 


                  for x in range(2, 6):  
                  print(x)

Мысал

Тізбектің қадамын көрсетіңіз: 


                    for x in range(2, 30, 3):  
                  print(x)

Else in For

else кілт сөзі цикл аяқталғаннан кейін орындалатын код блогын анықтайды.

Мысал

0-ден 5-ке дейінгі сандарды басып шығарыңыз және цикл аяқталған кезде хабарлама көрсетіңіз: 


                for x in range(6):  
                  print(x)  
                else:  
                  print("Цикл аяқталды!")

Ескерту: Егер цикл break операторымен тоқтатылса, else блогы орындалмайды.

Мысал

x 3-ке тең болғанда циклды тоқтатыңыз және else блогында не болатынын қараңыз: 


                for x in range(6):  
                if x == 3:  
                 break  
                print(x)  
                else:  
                  print("Цикл аяқталды!")

Ішкі циклдар

Ішкі цикл — бұл басқа циклдың ішіндегі цикл.
"Ішкі цикл" "сыртқы циклдың" әр итерациясы үшін бір рет орындалады.

Мысал

Әр жеміс үшін әр сын есімді шығарыңыз: 


                adj = ["қызыл", "үлкен", "дәмді"]  
                fruits = ["алма", "банан", "шие"]  
                
                for x in adj:  
                  for y in fruits:  
                    print(x, y)

Pass операторы

for циклы бос болмауы керек, бірақ егер қандай да бір себептермен цикл мазмұнсыз болса, қате жібермеу үшін pass операторын қосыңыз.

Мысал 


              for x in [0, 1, 2]:  
                  pass
