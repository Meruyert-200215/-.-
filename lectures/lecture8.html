8-дәріс: Python Тізімдер

Тізімдер дегеніміз не?

Тізімдер бірнеше элементтерді бір айнымалыда сақтауға арналған. Python-да тізімдер мәліметтер жиынтығын сақтайтын төрт кірістірілген деректер түрінің бірі болып табылады. Қалғандары: Кортеж,Жиын, және Сөздік .
Тізімдерді квадрат жақшалар арқылы жасауға болады:

Мысал 


                thislist = ["алма", "банан", "шие"]
                print(thislist)

Тізім элементтері

• Тізім элементтері реттелген, өзгермелі және қайталанатын мәндерді қолдайды. .
• Тізім элементтері индекс бойынша белгіленеді. Бірінші элементтің индексі - [0], екіншісі - [1], т.с.с.

Тізімдер реттілігі

Тізімдер реттелген болғандықтан, элементтердің реттілігі сақталады және оларды өзгертуге болмайды. Егер жаңа элемент қосылса, ол тізімнің соңына орналастырылады.

Тізімді өзгерту мүмкіндігі

Тізімдер өзгермелі, яғни оларды жасағаннан кейін де элементтерін өзгертуге, қосуға немесе жоюға болады.

Элементтердің қайталануы

Тізімдерде бірдей мәні бар элементтер болуы мүмкін, себебі олар индекстер арқылы айқындалады

Мысал:


    thislist = ["алма", "банан", "шие", "алма", "шие"]
    print(thislist)

Тізімнің ұзындығы

Тізімдегі элементтердің санын анықтау үшін len() функциясы қолданылады.

Мысал: 


                thislist = ["алма", "банан", "шие"]
                print(len(thislist))

Тізім элементтерінің мәліметтер түрлері

Тізім элементтері кез келген мәліметтер түріне ие болуы мүмкін:

Мысал: 


                list1 = ["алма", "банан", "шие"]  # Жолдар
                list2 = [1, 5, 7, 9, 3]           # Сандар
                list3 = [True, False, False]      # Булев мәндер

Тізім әртүрлі мәліметтер түрлерін де қамтуы мүмкін:

Мысал: 


            list1 = ["abc", 34, True, 40, "male"]

type() функциясы

Python-да тізімдер 'list' деректер түріне жатады.

Мысал: 


                mylist = ["алма", "банан", "шие"]
                print(type(mylist))

list() конструкторы

Тізімді жасау үшін list() конструкторын пайдалануға болады:

Мысал: 


            thislist = list(("алма", "банан", "шие"))  # қос жақшаға назар аударыңыз
            print(thislist)

Python-дағы жиынтық деректер түрлері

Python-да мәліметтер жиынтығын сақтауға арналған төрт деректер түрі бар:

    1. Тізім (List): Ретті және өзгермелі. Қайталанатын элементтерді қолдайды.
    2. Кортеж (Tuple): Ретті, бірақ өзгермейтін. Қайталанатын элементтерді қолдайды.
    3. Жиын (Set): Реттелмеген, өзгермейтін (элементтері өзгермейді), индекстелмеген. Қайталанатын элементтер жоқ.
    4. Сөздік (Dictionary): Ретті (Python 3.7-нұсқасынан бастап) және өзгермелі. Қайталанатын элементтер жоқ.

Әр деректер түрінің қасиеттерін түсіну деректерді дұрыс ұйымдастыруға және бағдарламаның тиімділігін арттыруға көмектеседі.

Тізімді әліпби немесе сандар бойынша сұрыптау

sort() әдісі әдепкі бойынша тізімді әліпби немесе сандық өсу ретімен сұрыптау үшін қолданылады.

Мысал (әліпби бойынша): 


                thislist = ["апельсин", "манго", "киви", "ананас", "банан"]
                thislist.sort()
                print(thislist)
                # Нәтиже: ['ананас', 'апельсин', 'банан', 'киви', 'манго']

Мысал (сандар бойынша): 


                thislist = [100, 50, 65, 82, 23]
                thislist.sort()
                print(thislist)
                # Нәтиже: [23, 50, 65, 82, 100]

Кему ретімен сұрыптау

reverse=True параметрін пайдаланып, тізімді кему ретімен сұрыптауға болады.

Мысал (кему ретімен әліпби бойынша): 


            thislist = ["апельсин", "манго", "киви", "ананас", "банан"]
             thislist.sort(reverse=True)
             print(thislist)
             # Нәтиже: ['манго', 'киви', 'банан', 'апельсин', 'ананас']

Мысал (кему ретімен сандар бойынша): 


            thislist = [100, 50, 65, 82, 23]
            thislist.sort(reverse=True)
            print(thislist)			
             # Нәтиже: [100, 82, 65, 50, 23]

Сұрыптауды өз функцияңызбен баптау

key параметрін пайдаланып, өз функцияңыз арқылы сұрыптауға болады.

Мысал: Сандарды 50-ге жақындығына қарай сұрыптау: 


            def myfunc(n):
                 return abs(n - 50)
                
            thislist = [100, 50, 65, 82, 23]
            thislist.sort(key=myfunc)
            print(thislist)
            # Нәтиже: [50, 65, 23, 82, 100]

Тіркелімді елемейтін сұрыптау

Әдепкі бойынша sort() әдісі үлкен және кіші әріптерді ажыратады. Үлкен әріптер алдымен сұрыпталады.

Мысал (тіркелімді ескеретін сұрыптау): 


            thislist = ["банан", "Алма", "Киви", "шие"]
                thislist.sort()
                print(thislist)
                # Нәтиже: ['Алма', 'Киви', 'банан', 'шие']

Тіркелімді елемейтін сұрыптау үшін str.lower функциясын пайдаланыңыз: 


                thislist = ["банан", "Алма", "Киви", "шие"]
                thislist.sort(key=str.lower)
                print(thislist)
                # Нәтиже: ['Алма', 'банан', 'шие', 'Киви']

Кері ретпен сұрыптау

Тізімді ағымдағы ретін өзгерткіңіз келсе, reverse() әдісін қолданыңыз.

Мысал: 


                thislist = ["банан", "Алма", "Киви", "шие"]
                thislist.reverse()
                print(thislist)
                # Нәтиже: ['шие', 'Киви', 'Алма', 'банан']
