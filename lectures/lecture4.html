4-дәріс: Python-дағы сандар

Python-да үш сандық түр бар:
• int
• float
• complex

Айнымалылар сандық түрлерге оларға мәнді тағайындаған кезде жасалады:

Мысал


 x = 1  # int
y = 2.8  # float
z = 1j  # complex

Кез келген объектінің түрін тексеру үшін Python-да type() функциясын пайдаланыңыз:

Мысал


print(type(x))
print(type(y))
print(type(z))

Бүтін сандар (Int)

Int - немесе бүтін сан, оң немесе теріс бүтін санды, ондық бөлшексіз, шектеусіз ұзындықпен білдіреді.

Мысал


x = 1
y = 35656222554887711
z = -3255522
    
print(type(x))
print(type(y))
print(type(z))

Бөлшек сандар (Float)

Float - немесе бөлшек сан, оң немесе теріс болуы мүмкін және бір немесе бірнеше ондық бөлшекке ие болады.

Мысал


x = 1.10
y = 1.0
z = -35.59

print(type(x))
print(type(y))
print(type(z))

Бөлшек сандар 10-ның дәрежесін білдіретін "e" әрпі арқылы көрсетілуі мүмкін.

Мысал


x = 35e3
y = 12E4
z = -87.7e100
    
print(type(x))
print(type(y))
print(type(z))

Күрделі сандар (Complex)

Complex - немесе күрделі сандар, мнималық бөлігін көрсету үшін "j" әрпін пайдаланады.

Мысал


    x = 3+5j
    y = 5j
    z = -5j
    
    print(type(x))
    print(type(y))
    print(type(z))

Типті түрлендіру

Бір типті екінші типке келесі әдістерді пайдаланып түрлендіруге болады: int(), float(), complex().

Мысал 


x = 1  # int
y = 2.8  # float
z = 1j  # complex

# int-ті float-қа түрлендіру:
a = float(x)

# float-ты int-ке түрлендіру:
b = int(y)

# int-ті complex-ке түрлендіру:
c = complex(x)

print(a)
print(b)
print(c)

print(type(a))
print(type(b))
print(type(c))

Ескерту: Күрделі сандарды басқа типке түрлендіру мүмкін емес.

Кездейсоқ сандар

Python-да кездейсоқ сандар жасау үшін random() функциясы жоқ, бірақ кездейсоқ сандар жасау үшін random модулін пайдалануға болады:

Мысал 


            import random

            print(random.randrange(1, 10))
