1-дәріс: Python-ға кіріспе

Python дегеніміз не?

Python - танымал бағдарламалау тілі. Ол Guido van Rossum жасалып, 1991 жылы шығарылған.
Python келесі салаларда қолданылады:

    • веб-әзірлеу (серверлік бөлігі),
    • бағдарламалық қамтамасыз етуді әзірлеу,
    • математика,
    • жүйелік сценарийлер.

Python не істей алады?

Python серверде веб-қосымшалар жасау үшін қолданыла алады.
Python бағдарламалық қамтамасыз етумен біріктіріліп, жұмыс процестерін құра алады.
Python дерекқор жүйелеріне қосылып, файлдарды оқып, өзгерте алады.
Python үлкен деректерді өңдеу және күрделі математикалық операцияларды орындау үшін қолданыла алады.

Неге Python?

• Python әртүрлі платформаларда жұмыс істейді (Windows, Mac, Linux, Raspberry Pi және т.б.).
• Python синтаксисі бағдарламашыларға кодты басқа бағдарламалау тілдеріне қарағанда аз жолдармен жазуға мүмкіндік береді.
• Python синхронды аударма жүйесінде жұмыс істейді, бұл код жазылғаннан кейін бірден орындала алатынын білдіреді. Бұл прототип жасауды өте жылдам етуге мүмкіндік береді.
• Python-ды процедуралық, объектіге бағытталған немесе функционалдық тәсілдермен қолдануға болады.

Python синтаксисі басқа бағдарламалау тілдерімен салыстырғанда қандай?

• Python кодты оқу жеңілдігін ескере отырып әзірленген және ағылшын тіліндегі математикалық синтаксиске ұқсас.
• Python команданы аяқтау үшін жаңа жолдарды пайдаланады, ал басқа бағдарламалау тілдерінде жиі үтір немесе жақшалар қолданылады.
• Python әрекеттердің аясын анықтау үшін бос орындарды пайдаланып, шегініс қолданады; бұл циклдердің, функциялардың және сыныптардың көлемін анықтайды. Басқа бағдарламалау тілдерінде бұл мақсатта әдетте фигуралық жақшалар пайдаланылады.
